# Plateforme-de-Surveillance-de-la-Biodiversit-A-roportuaire
Chatbot IA (Agno) pour la gestion du risque faunique en aéroport. Analyse en temps réel la biodiversité et les incidents (bird strikes) en fusionnant PostGIS (géo), Neo4j (graphe) et RAG. Inclut un workflow d'ingestion et une carte de monitoring interactive.
